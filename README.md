# SQL Task 3 – Data Analyst Internship

## Objective
To practice basic SQL operations including filtering, sorting, grouping, aggregation, and exporting results.

## Tools Used
- MySQL / PostgreSQL
- MySQL Workbench / DBeaver
- CSV Dataset (Superstore Sales)

## Queries Performed

1. Viewed dataset using SELECT *
2. Filtered data using WHERE clause
3. Sorted sales data using ORDER BY
4. Used aggregation functions SUM, AVG, COUNT
5. Grouped results using GROUP BY
6. Filtered grouped results using HAVING
7. Used BETWEEN for date filtering
8. Used LIKE for pattern matching
9. Found top 5 customers by total sales

## Key Learnings
- WHERE filters rows before grouping
- HAVING filters results after aggregation
- GROUP BY summarizes data
- ORDER BY sorts the output
- NULL values are ignored in SUM and AVG
